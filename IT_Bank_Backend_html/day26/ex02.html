Tag 설명 예시
html HTML 문서의 시작과 끝 위치를 나타냄
head 머릿글, 문서에 전반적으로 적용되는 설정
body 본문, 브라우저에서 나타날 화면을 작성

태그의 구조 
          <여는 태그 속성1 = "값" 속성2 ="값"> 
              글자 및 다른 태그 내용
          </닫는태그>

태그이름과 속성은 띄어쓰기로 구분한다.

속성이름과 속성값은 =로 구분되고, 띄어쓰기를 사용하지 않는다.

속성값은 쌍따옴표(")로 묶어야 한다.

여는 태그와 닫는 태그의 깊이(depth)는 같아야 함.

여는 태그와 닫는 태그의 이름은 같아야함.

태그 종류에 따라 용도가 다르고, 사용할 수 있는 속성이 다르다.


"
